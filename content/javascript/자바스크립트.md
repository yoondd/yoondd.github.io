## immutable 
변경이 불가능한 값

```js
const a = 1;
a = 2;
```

리터럴이라서 변경이 불가능하다.



## mutable
변경가능한 값

```js
const a2 = { a: null, b: null }
a2.a = 1;
a2.b = 2;
```
