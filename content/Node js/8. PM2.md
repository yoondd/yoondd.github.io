
서버가 꺼지면 다시 실행시켜주는 기능도 하고있고,
파일이 수정되는지 관찰하고있다가 수정되었으면 자동으로 껐다켜주는 기능도 하고있다

# 설치

```
sudo npm install pm2 -g
```


# 실행

```
pm2 start main --watch
```
- watch기능을 넣어서 파일수정시 자동으로 서버를 재시동하도록 설정할 수 있다

# 현재 pm2로 실행하고있는 프로그램 보기

```
pm2 monit
```


# 현재 실행중인 프로세스 확인

```
pm2 list
```


# 로그보기(에러확인)

```
pm2 log
```


# 종료

```
pm2 stop main
```
