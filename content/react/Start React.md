3. 리액트 설치하기
	npx가 아니라도 괜찮지만, 리액트가 원하는 설정을 빠르게 할 수 있어서 많은 사용자들이 npx를 이용한다.
	프로젝트 이름은 추후에 깃허브에 편하게 배포하기위해 간단하게 만들 수 있다
	나의 프로젝트: FRONT_CLASS > class1
```bash
npx create-react-app <나의프로젝트이름>
```
