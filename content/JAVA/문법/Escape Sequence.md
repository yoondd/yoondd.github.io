자바의 이스케이프 문자는 문자열 내에서 특수한 기능을 수행하는 문자로, 보통 역슬래시(`\`)와 특정 문자를 조합해 사용한다. 

|이스케이프 문자|의미|
|---|---|
|`\n`|새 줄 (줄 바꿈)|
|`\t`|탭 (수평 탭)|
|`\r`|캐리지 리턴 (커서를 줄 맨 앞으로 이동)|
|`\b`|백스페이스 (커서를 한 칸 뒤로 이동)|
|`\f`|폼 피드 (프린터에서 페이지 넘김)|
|`\'`|작은 따옴표 출력|
|`\"`|큰 따옴표 출력|
|`\\`|역슬래시 출력|

엄청 어려운 의미는 아니므로 많이 사용하는 것만 정리해준다.
