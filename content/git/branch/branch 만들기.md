## 만들어보자

```bash
git branch exp
```
 
이렇게 만들고나서 다시한번 브랜치를 쳐보면 별표시가 어디에 있는가? main에 있지?
내가 지금 main에 있기 때문이다.


## 브랜치를 이동해보자

```js
git checkout exp
```

이동한 순간까지는 git log와 파일들의 상태를 함께 공유했지만 나누어지는 순간 각자 관리하게 되는거다
