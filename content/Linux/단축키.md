| 터미널 내 새 탭 열기     | Ctrl + Shift + T |
| 터미널 내 새 창 열기     | Ctrl + Shift + N |
| 터미널 종료           | Ctrl + Shift + Q |
| 터미널 내 문자열 검색     | Ctrl + Shift + F |
| 복사               | Shift + Ctrl + C |
| 붙여넣기             | Shift + Ctrl + V |
| 화면 초기화(clear)    | Ctrl + L         |
| 실행 중인 프로그램 중지    | Ctrl + C         |
| 실행 중인 프로그램 일시 정지 | Ctrl + Z         |
